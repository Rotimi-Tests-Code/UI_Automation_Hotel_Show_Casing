# UI_Automation_Hotel_Show_Casing
Mission: End to end test automation project for validating hotel booking flow using Playwright and TypeScript


# Tech Stack 

- Playwright 
- TypeScript
- Node.js 


# Install Playwright 
- npm install 
- npx playwright test 

# Project Structure 
- tests/ #bookinghotel.spec.ts 
- pages/ #bookNow.ts, homePage.ts
- gitignore
- package-lock.json
- package.json
- playwright.config.ts 
- READ.ME.md 

# Explanation
- Run the tests using playwright's test runner 
- usable command npx playwright test (This runs all tests on playwright)
- This is a basisc E2E automation framework using Playwright
